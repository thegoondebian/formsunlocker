google forms unlocker

this user script allows you to bypass the locked mode in google forms. it lets you access forms that are restricted to managed chromebooks, even if you're using a different device.

features

bypass google forms locked mode
works without needing a chromebook
allows you to fill out forms freely

requirements

tampermonkey or greasyfork installed in your browser
user agent spoofer if you're getting locked mode errors

installation

install tampermonkey or greasyfork extension in your browser if you haven't already.
copy the script in this repository
load it into the userscript loader

usage

open the google form you want to access.
if the form is locked, a bypass button will appear at the bottom of the form.
click the bypass button to access the form.
